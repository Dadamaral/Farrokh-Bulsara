
# Farrokh-Bulsara

O repositório contará com um index introdutório, com informações cruciais acerca do tema e que levam a posteriormente a outras páginas mais detalhadas. Barra de navegação, ilustrações gráficas e tema das páginas serão apresentados na primeira página. 
As páginas detalhadas serão acerca de temas específicos da biografia, contemplando arquivos de áudio, imagens, formulário e interações afim de enriquecer o contato do usuário com a vida de Farrokh Bulsara.
